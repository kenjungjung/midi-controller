> GPIO番号はすべて仮。PCB設計時に確定する。
> アナログ入力はADC1（GPIO1–10）のみ使用。

## デジタル入力（PCF8574 I2C GPIOエクスパンダ経由）

ボタンはすべて PCF8574 経由で読み取る。ESP32-S3 の GPIO を消費しない。
SDA/SCL は OLED（SSD1306）と同じ I2C バスを共有。

### PCF8574 #1（I2Cアドレス: 0x20） — MIDI ボタン

| PCF8574ピン | 対象 | 備考 |
|------------|------|------|
| P0 | ボタン 1 | プルアップ内蔵、アクティブLow |
| P1 | ボタン 2 | プルアップ内蔵、アクティブLow |
| P2 | ボタン 3 | プルアップ内蔵、アクティブLow |
| P3 | ボタン 4 | プルアップ内蔵、アクティブLow |
| P4 | ボタン 5 | プルアップ内蔵、アクティブLow |
| P5 | ボタン 6 | プルアップ内蔵、アクティブLow |
| P6 | ボタン 7 | プルアップ内蔵、アクティブLow |
| P7 | ボタン 8 | プルアップ内蔵、アクティブLow |

### PCF8574 #2（I2Cアドレス: 0x21） — バンク切り替え

| PCF8574ピン | 対象 | 備考 |
|------------|------|------|
| P0 | バンク切り替え 1（Bank1） | プルアップ内蔵、アクティブLow |
| P1 | バンク切り替え 2（Bank2） | プルアップ内蔵、アクティブLow |
| P2–P7 | 未使用（予備） | — |

> I2Cアドレスは A0/A1/A2 ピンで設定。0x20 = A2/A1/A0 すべてLow、0x21 = A0のみHigh。

## KiCad設計時の注意事項

- **GPIO19/20禁止**: ESP32-S3内蔵USB（D-/D+）専用。他の用途に使用不可。
- **WS2812B電源**: 36個 × 60mA × 25%輝度 ≈ 540mA。USB-C給電（900mA〜）で賄える。ソフトウェアで `MAX_BRIGHTNESS = 64/255` に制限。
- **WS2812Bレベル変換**: データ線は3.3V出力→5V入力。74AHCT1G125等でレベル変換推奨。
- **プルアップ抵抗**: ボタン系は10kΩでVCC（3.3V）にプルアップ。
- **デカップリングコンデンサ**: 各IC電源ピンに100nFを配置。
